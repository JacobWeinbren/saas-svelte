# Alert

Used to communicate a state that affects a system, feature or page.

<Canvas of={AlertStories.Basic} />

## Anatomy

The Alert component is a single unit that accepts properties for status, variant, and content.

```svelte
import { Alert } from '$saas/alert';

<Alert status="info" title="Alert Title">
  Description of the alert goes here.
</Alert>

```

## Examples

Use the `children` snippet to provide additional context to the alert. This will be displayed below the alert message.

<Canvas of={AlertStories.WithDescription} />

## Status

Change the status of the alerts by passing the `status` prop. This affects the color scheme and icon used. Alert supports `error`, `success`, `warning`, and `info` statuses.

<Canvas of={AlertStories.WithStatus} />

## Variants

Use the variant prop to change the visual style of the alert. Values can be `subtle`, `solid`, `surface`, or `outline`.

<Canvas of={AlertStories.WithVariants} />

## Custom Icon

Use the `icon` prop to pass a custom icon to the alert. This will override the default icon for the alert status.
