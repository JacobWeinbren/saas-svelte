# Accordion

Used to show and hide sections of related content on a page

<Canvas of={AccordionStories.Basic} />

## Anatomy

The Accordion component uses Ark UI primitives for accessibility, state management, and keyboard navigation.

**Simple API (using items prop):**

```svelte
import { Accordion } from '$saas/accordion';

const items = [
  { value: 'a', title: 'First Item', content: 'Some value 1...' },
  { value: 'b', title: 'Second Item', content: 'Some value 2...' },
  { value: 'c', title: 'Third Item', content: 'Some value 3...' },
];

<Accordion {items} collapsible defaultValue={['b']} />
```

**Composition API (for full control):**

```svelte
import { AccordionComposition } from '$saas/accordion';

<AccordionComposition.Root collapsible defaultValue={['b']}>
  <AccordionComposition.Item value="a">
    <AccordionComposition.ItemTrigger>First Item</AccordionComposition.ItemTrigger>
    <AccordionComposition.ItemContent>Some value 1...</AccordionComposition.ItemContent>
  </AccordionComposition.Item>

  <AccordionComposition.Item value="b">
    <AccordionComposition.ItemTrigger>Second Item</AccordionComposition.ItemTrigger>
    <AccordionComposition.ItemContent>Some value 2...</AccordionComposition.ItemContent>
  </AccordionComposition.Item>

  <AccordionComposition.Item value="c">
    <AccordionComposition.ItemTrigger>Third Item</AccordionComposition.ItemTrigger>
    <AccordionComposition.ItemContent>Some value 3...</AccordionComposition.ItemContent>
  </AccordionComposition.Item>
</AccordionComposition.Root>
```
