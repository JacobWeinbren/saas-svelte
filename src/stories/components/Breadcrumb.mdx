# Breadcrumb

## Used to display a page's location within a site's hierarchical structure.

<Canvas of={BreadcrumbStories.Basic} />

## Anatomy

The Breadcrumb component uses a compound component pattern with a Root, Link, CurrentLink, and optional Ellipsis.

```tsx
<script>
	import {
		BreadcrumbRoot,
		BreadcrumbLink,
		BreadcrumbCurrentLink
	} from "$saas/components/breadcrumb";
</script>

<BreadcrumbRoot separator="/">
	<BreadcrumbLink href="#">Workspace</BreadcrumbLink>
	<BreadcrumbLink href="#">Projects</BreadcrumbLink>
	<BreadcrumbCurrentLink>Settings</BreadcrumbCurrentLink>
</BreadcrumbRoot>
```

## Routing Library

Use the standard `href` prop on `BreadcrumbLink` for navigation. For SvelteKit, you can use the standard anchor behaviour.

```tsx
<script>
	import {
		BreadcrumbRoot,
		BreadcrumbLink,
		BreadcrumbCurrentLink
	} from "$saas/components/breadcrumb";
</script>

<BreadcrumbRoot>
	<BreadcrumbLink href="/docs">Docs</BreadcrumbLink>
	<BreadcrumbLink href="/docs/components">Components</BreadcrumbLink>
	<BreadcrumbCurrentLink>Breadcrumb</BreadcrumbCurrentLink>
</BreadcrumbRoot>
```
