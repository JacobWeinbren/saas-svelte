# Select (Native)

## Used to pick a value from predefined options.

<Canvas of={NativeSelectStories.Basic} />

## Anatomy

```tsx
<script>
	import { NativeSelect } from "$saas/components/native-select";
</script>

<NativeSelect>
	<option value="1">Option 1</option>
	<option value="2">Option 2</option>
</NativeSelect>
```
