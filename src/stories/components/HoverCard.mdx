# Hover Card

## Used to display content when hovering over an element.

[Ark UI](https://ark-ui.com/docs/components/hover-card)

<Canvas of={HoverCardStories.Basic} />

## Anatomy

```tsx
<script>
	import { HoverCard } from "$saas/components/hover-card";
</script>

<HoverCard.Root>
	<HoverCard.Trigger />
	<HoverCard.Content>
		<HoverCard.Arrow />
	</HoverCard.Content>
</HoverCard.Root>
```

## Accessibility

### Keyboard Support

| Key | Description |
|-----|-------------|
| `Enter` / `Space` | Opens or closes the hover card when the trigger is focused |
| `Escape` | Closes the hover card |

### ARIA Attributes

- The trigger element has `aria-expanded` set to reflect the open state
- The content has `role="tooltip"` to identify it as supplementary information

HoverCard is best suited for supplementary information. For critical content, consider using a Popover or Dialog instead.
