# Field

## Used to add labels, help text, and error messages to form fields.

<Canvas of={FieldStories.Basic} />

## Anatomy

```html
<script>
	import { Field } from "$saas/components/field";
	import { Input } from "$saas/components/input";
</script>

<Field.Root>
	<Field.Label />
	<Input />
	<Field.HelperText />
	<Field.ErrorText />
</Field.Root>
```
