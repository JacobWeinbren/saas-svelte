# Menu

## Used to create an accessible dropdown menu.

<Canvas of={MenuStories.Basic} />

## Anatomy

```svelte
<script>
	import {
		MenuRoot,
		MenuButton,
		MenuContent,
		MenuItem
	} from "$saas/components/menu";
</script>

<MenuRoot>
	<MenuButton />
	<MenuContent>
		<MenuItem value="item" />
	</MenuContent>
</MenuRoot>
```
