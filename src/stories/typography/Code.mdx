# Code

## Used to display inline code snippets with syntax highlighting.

<Canvas of={CodeStories.Basic} />

## Anatomy

```html
<script>
	import { Code } from "$saas/typography/code";
</script>
```

```html
<code>console.log("Hello, world!")</code>
```

## Examples

### Sizes

Use the `size` prop to change the size of the code element. You can set the value to `xs`, `sm`, `md`, or `lg`.

<Canvas of={CodeStories.Sizes} />

### Variants

Use the `variant` prop to change the visual style of the code element. You can set the value to `solid`, `outline`, `subtle`, or `surface`.

<Canvas of={CodeStories.Variants} />

### Colors

Use the `color` prop to change the colour palette of the code element. It supports all standard Tailwind colors.

<Canvas of={CodeStories.Colors} />
