# Highlight

Used to highlight substrings of a text.

<Canvas of={HighlightStories.Basic} />

## Usage

```tsx
<script>
	import { Highlight } from "$saas/typography/highlight";
</script>

<Highlight query="Hello" text="Hello World" />
```

## Examples

### Multiple

Pass an array of strings to the `query` prop to highlight multiple substrings.

<Canvas of={HighlightStories.Multiple} />

```tsx
<Highlight
	query={["spotlight", "emphasise", "accentuate"]}
	text="With the Highlight component, you can spotlight, emphasise and accentuate words."
	colour="teal"
/>
```

### Custom style

Use the `variant` prop to customise the style of the highlighted text.

<Canvas of={HighlightStories.CustomStyle} />

```tsx
<Highlight
	query="component"
	variant="plain"
	text="With the Highlight component, you can spotlight words."
/>
```

### Search query

Use the highlight component to highlight search query results.

<Canvas of={HighlightStories.SearchQuery} />

```tsx
<script>
	const query = "spot";
	const results = ["Spotlight bulb", "Spot cleaner", "Spot ceiling"];
</script>

<Stack>
	<Text>Search result for: spot</Text>
	<Stack>
		{#each results as item}
			<Text>
				<Highlight ignoreCase {query} variant="plain" text={item} />
			</Text>
		{/each}
	</Stack>
</Stack>
```
