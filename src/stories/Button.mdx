# Button

Used to trigger an action or event.

<Canvas of={ButtonStories.Basic} />

## Anatomy

```html
<script>
	import { Button } from "$saas/button";
	import { ButtonGroup } from "$saas/ButtonGroup.svelte";
</script>

<button variant="solid" colorPalette="blue">Click me</button>
```

Examples
Sizes
Use the size prop to change the size of the button. You can set the value to xs, sm, md, lg, or xl.

<Canvas of={ButtonStories.Sizes} />

Variants
Use the variant prop to change the visual style of the Button. You can set the value to solid, subtle, outline, ghost, plain, surface, or glass.

<Canvas of={ButtonStories.Variants} />

Icon
Use icons within a button by passing them as children.

<Canvas of={ButtonStories.WithIcon} />

Color
Use the colorPalette prop to change the color of the button. It supports all standard Tailwind colors.

<Canvas of={ButtonStories.Colors} />

Loading
Use the loading and loadingText props to show a loading spinner.

<Canvas of={ButtonStories.Loading} />

Button Group
Use the ButtonGroup component to group buttons together.

<Canvas of={ButtonStories.Group} />
